A = empty Queue of nodes found
size = 0
size1 = sizeof(G)
Traverse(G)
    Put node (r) in A
    While A is not empty do
        Take a node c from A
        If c isn’t marked as found then
            Mark c as found
            size = size + 1
            for each neighbor x to c
                Put x in A
        end if
    end while
end Traverse
if size = size1 return true else return false

This algorithm will search through the graph and put each non-found node visited into the queue. From here we will dequeue it and follow the paths to the neighbors of the node we just removed from the queue. If we have already visited the nodes then we will not mark them and we won’t do anything with them after dequeueing. This prevents us from adding in their neighbors multiple times and preventing a never ending loop. By adding one every time we take a node from the queue to size we will get the total amount of nodes we have visited throughout our search. Once we know the amount of nodes we have searched we will compare that to the size of the graph. If they are equal then the graph will visit every vertex, if it’s not then every vertex is not visited.
Proof of correctness.
